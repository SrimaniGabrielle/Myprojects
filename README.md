# LLM Hallucination Detection using Retrieval-Based Verification
This project implements a retrieval-based hallucination detection system for Large Language Models (LLMs). The system generates answers using an LLM and verifies the factual correctness of each claim by retrieving supporting evidence from a knowledge base derived from the Climate-FEVER fact verification dataset.

By comparing semantic similarity between generated claims and retrieved evidence, the system identifies whether the response is supported, partially supported, or potentially hallucinated.

The goal of this project is to demonstrate practical implementation of RAG-style retrieval, vector search, and explainable AI techniques to improve the reliability of generative AI systems.
